# 🚀 Personal Developer Portfolio

> A modern, responsive, single-file portfolio website — built with HTML, CSS & JavaScript.  
> Designed for developers to showcase projects, skills, and experience to recruiters.

Live Demo = https://portfolio-rose-nine-18.vercel.app/
GitHub = https://github.com/davidsonu00/Portfolio

---



## ✨ Features

- 🌙 **Dark / Light mode** toggle
- 📱 **Fully responsive** — mobile, tablet, desktop
- ⚡ **Zero dependencies** — no npm install, no build step
- 🎯 **Project filtering** — filter by Web, Python, AI
- 🎞️ **Scroll animations** — skill bars, timeline, section reveals
- 📬 **Contact form** with success/error notifications
- 🔝 Scroll-to-top button
- 🔍 SEO meta tags included
- ♿ ARIA accessible markup

---

## 🗂️ Sections

| Section | Description |
|---|---|
| **Hero** | Name, role, tagline, CTA buttons, stats |
| **About** | Bio, career goals, personality tags |
| **Skills** | Progress bars for languages/frontend/backend, badge grid for tools |
| **Projects** | Card grid with live demo + GitHub links, category filter |
| **Experience** | Animated timeline — education, internships, certifications |
| **Resume** | One-click PDF download |
| **Contact** | Form + social links (GitHub, LinkedIn, Email, Twitter) |

## 🧰 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Styling, animations, dark/light theme via CSS variables |
| Vanilla JavaScript | Interactivity, scroll effects, form handling |
| Google Fonts | Syne · DM Mono · Instrument Serif |
| IntersectionObserver API | Scroll-triggered animations |
| EmailJS *(optional)* | Contact form email delivery |
